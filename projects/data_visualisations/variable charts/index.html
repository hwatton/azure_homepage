Variable average graphs

*Note: this page is best viewed landscape or on a desktop/laptop. Thanks for taking a look.

The data in these graphs represents the daily recorded deaths from COVID-19, per country.
These graphs can be altered using the slider on the right.
The line represents an average of between 2 and 50 days data.
By increasing the averaging sample size, we can more clearly identify the historic trends in the data.


Line data represents the average of the 10 previous days.




This series of graphs has been produced to demonstrate some data visualisation techniques with d3.js.
The respective data have been scaled such that the shape of the graphs are similar.
ie: the scales are different with respect to each country's available data.
As such it is important to note that while the trends are easily visible, the rate
of change at any one point is not necessarily given fairly by the gradient of the line.
For example, Iran's data shows a dramatic increase towards the end of the dataset, but given
that the absolute numbers are so much smaller than, for example, the US, the increase would
well explained by only a small % deviation within the numbers.

The data is sourced from www.ecdc.europa.eu.
I have made quite a few graphs from live json API's but as I don't monitor the source regularly, I have chosen to use a static csv file for this visualisation.
